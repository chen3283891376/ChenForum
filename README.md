## 准备开发

1. 下载项目 `git clone https://github.com/chen3283891376/ChenForum.git`
2. 切换到项目文件夹
3. 安装依赖项 `npm install`
4. 开发时最好开着开发服务器，让 React Router 初始化一下 Router 的 types

## npm脚本

- `npm run dev`: 打开开发服务器
- `npm run typecheck`: 类型检测
- `npm run format`: 格式化
